# Dotnet Code Intelligence

Read-only CLI tool for semantic analysis of C#/.NET solutions using Roslyn.

## Goal
Help developers and AI agents understand legacy and mixed .NET solutions faster:
- find symbols
- find references
- find implementations
- analyze impact of changes
- inspect solution structure

## Status
MVP in progress.

## Planned commands
- solution-summary
- find-symbol
- find-references
- find-implementations
- analyze-impact
- find-registrations
- trace-callers
